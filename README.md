# Football Analytics: Expected Goals (xG) Dashboard

A sports analytics project for the AQX Hackathon analyzing expected goals (xG) metrics in football/soccer.

## Project Overview

Expected Goals (xG) is a key metric in modern football analytics that measures the quality of chances created and conceded. This dashboard provides:

- **Team xG Analysis**: Compare teams by expected goals for/against
- **Historical Trends**: Track xG performance over time
- **Actionable Insights**: Identify underperforming teams and missed conversion opportunities
- **Clear Visualizations**: Interactive charts and dashboards
