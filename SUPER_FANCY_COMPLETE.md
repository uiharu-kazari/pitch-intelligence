```
Primary: #0f1a3a to #3d63c4 (deep to bright blue)
Success: #10b981 (green for positive metrics)
Warning: #f59e0b (orange for alerts)

```

### Typography System
```
Headings: 54px, 30px, 24px, 20px
Body: 16px, 14px, 12px
Font: System fonts (optimal performance)

### Animations
```

Base: 300ms (smooth transitions)
Slow: 500ms (emphasis effects)

```

```
1200px: Large desktop (full grid)
768px: Tablet (2-column or single)
480px: Mobile (single column, optimized)
